
## Layer詳細

### Layer 0：環境（OS更新のトリガー）

OSは思想や意志ではなく、**環境変化への適応反応**として書き換わります。

| 環境変数     | 変化の流れ                  | 影響                              |
|--------------|-----------------------------|-----------------------------------|
| メディア構造 | TV → SNS → AI拡散          | 動員速度が秒単位へ                |
| 経済構造     | 成長 → 停滞 → 不安定化      | 実利・安全が優先                  |
| 社会心理     | 希望 → 怒り → 疲労          | ideology fatigue（思想疲労）      |
| 情報速度     | 週 → 日 → 秒                | 感情の瞬間増幅                    |

> OSは「進化」ではなく**環境適応**です。

### Layer 1：入力（Raw Input）

社会に最初に現れるのは、まだ構造化されていない**感情エネルギー**。

- 怒り  
- 不安  
- 希望  
- 正義  
- 被害感覚  
- 理念  

**まだ構造ではない。ただの燃料です。**

### Layer 2：翻訳エンジン（Narrative Engine）

ここで感情は「物語」に圧縮されます。

- 善 vs 悪  
- 被害者 vs 加害者  
- 私たち vs 彼ら  
- 正義の側  

> 現実はここで**ストーリー化**されます。

### Layer 3：動員OS（Mobilization OS）

物語が「社会を実際に動かす方式」に変換される中核レイヤー。

| 世代          | OS名              | 動員燃料           | 特徴・時代                |
|---------------|-------------------|--------------------|---------------------------|
| 第1世代       | 理念OS            | 理想・正義・思想   | 長期運動・組織型（戦後〜80s） |
| 第2世代       | 共感OS            | 象徴・イメージ     | TV主導・瞬間動員（90s〜00s） |
| 第3世代       | 敵味方OS          | 怒り・対立・アイデンティティ | SNS分断・炎上拡散（2010s） |
| 第4世代       | 生活機能OS        | 実利・安全・体感   | 低温持続型（2020s〜）     |
| 第5世代（仮） | AI共鳴OS          | 感情最適化アルゴリズム | 超個別動員・体験政治（未来） |

> 選挙・NGO・SNS運動はすべて**このレイヤーで動いています**。

### Layer 4：制度フィルター（Institution Filter）

動員OSは制度によって歪められます。

- 資金構造  
- 支持層構造  
- メディア構造  
- 法制度  
- 組織の生存本能  

> 善悪ではなく**インセンティブの結果**として現実は歪む。

### Layer 5：均衡（System Equilibrium）

なぜ社会は変わらないのか？

- 支持者離脱の恐怖  
- 炎上回避  
- ナッシュ均衡  
- 生存インセンティブ  

> 誰も「最適解」を選ばない。古いOSでも動き続けます。

### Layer 6：出力（Visible Reality）

私たちが目にする現象はすべて**結果**です。

- 炎上  
- 分断  
- 政治ショー  
- NGOの選択的沈黙  
- メディアの奇妙な一致
